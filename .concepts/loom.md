### Loom

[Loom](https://generative.ink/posts/loom-interface-to-the-multiverse/) is an experimental writing interface created by Janus for generating, navigating, and visualizing branching “natural language multiverses” with large language models. It was designed as a way to preserve and explore multiple narrative branches during AI-assisted writing, rather than forcing a single linear history.

The system includes a read mode (showing one lineage as a coherent timeline), a visualize mode (displaying the branch structure as a tree), and node-level metadata for prompts, outputs, and model statistics. In Janus’s framing, Loom treats language models as multiverse generators and uses branching as a core creative workflow.
