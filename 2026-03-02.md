### 2026-03-02

Input is live now. It's an alternative GitHub frontend that implements a bunch of patterns that I've been interested in exploring for a while.

You can drop in any repo, and just replace github.com with input.md, to get a HackMD/Obsidian-style index of the files:

```
input.md/raykyri/homepage
```

It asks for minimal permissions (access to gists, and only repos that you authorize).

You can use it with just Gists, although if you want privacy you should connect your own GitHub repo.

![pasted-20260304-155047-tinc4n.png](./.assets/pasted-20260304-155047-tinc4n.png)

Input is stateless -- we persist no user data to our own database, just OAuth tokens and app installation tokens. This is intentional, as I want anyone to be able to fork and build off it as easily as possible.
